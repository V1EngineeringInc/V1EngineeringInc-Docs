    LowRider v4 Size Calculator¶

    Inputs¶

    Units¶

    Millimeter (mm)
    Inch

    Machine¶

    LowRider v4

    Usable Cutting Area¶

    mm X
    mm Y

    XZ Plate Thickness¶

    Shop Aluminum plates are 6.35mm (0.25”).

    mm XZ Plate

    Size Calculator¶

    Rail Lengths¶

    Length (mm) Qty Name
    2 X Rails
    1 Y Rail

    Belt Dimensions¶

    Length (mm) Qty Name
    1 Belt length along X
    2 Belt length along Y
    total length belts (all 3)

    Strut Plates¶

    Length (mm) Qty Name
    2 Strut length

    Table Size¶

    This output is the minimum table required. An extra 25-50mm (1”-2”) or more on each dimension is nice if you will be pushing it up against a wall or in a corner.

    Length (mm) Name
    X table size (width)
    Y table size (length)
